## 📁 文件结构

请确保你的文件夹结构如下：

```

├── index.html
├── assets/              # ⭐ 新建这个文件夹
│   ├── kay_happy.png    # 开心团子
│   ├── kay_smug.png     # 傲娇团子
│   ├── kay_sweat.png    # 流汗鸭
│   └── kay_elephant.png # 大象
├── libs/
│   └── face-api.min.js
└── models/
    └── ...
```

## 🎨 Kay的作品展示

### 如何使用：
1. 将Kay的4个作品放入 `assets/` 文件夹
2. 打开 FaceJam 应用
3. 点击 **"Kay"** 标签页（带💕图标）
4. 就能看到Kay的所有原创贴纸啦！

- **傲娇团子** (kay_smug.png)

## 🎯 新增的自定义贴纸功能

### 1. 本地存储 💾
- ✅ 上传的自定义贴纸会自动保存在浏览器本地
- ✅ 刷新页面后贴纸不会丢失
- ✅ 使用 localStorage 存储，永久保存

### 2. 删除功能 🗑️
**如何删除：**
1. 切换到"自定义"标签页
2. 鼠标悬停在要删除的贴纸上
3. 会出现红色的 ✕ 按钮（右上角）
4. 点击即可删除

**特点：**
- 悬停时才显示删除按钮（界面简洁）
- 删除前会确认（防止误操作）
- 删除后立即从本地存储中移除

## 🚀 快速开始

1. **准备文件**
   ```bash
   mkdir assets
   # 将4个png文件放入assets文件夹
   ```

2. **打开应用**
   - 直接打开 index.html
   - 或通过HTTP服务器访问

3. **查看Kay专区**
   - 点击"Kay"标签页
   - 选择喜欢的贴纸
   - 开始创作！
